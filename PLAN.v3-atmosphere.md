# Plan v3.0 - Atmosphere Rim Refinement Plan

## 文档定位
- 本文档是 **v3.0 视觉美化计划**，针对“地球周围蓝圈过重”的问题进行优化。
- 与历史计划分区：
  - `PLAN.md`：v1.0 功能建设计划
  - `PLAN.v2-debug.md`：v2.0 稳定性与可用性修复计划
  - `PLAN.v3-atmosphere.md`：v3.0 光晕视觉质量提升计划

## 1. 当前问题（2026-02-26）
- 地球外圈出现明显“蓝色描边感”。
- 大气层视觉过于均匀、厚重，影响真实感。
- 场景主体细节（地表与云层）被边缘高亮抢焦点。

## 2. 目标
- 把“硬蓝圈”优化为“薄、柔、分层”的大气边缘。
- 增强真实感：日侧亮、夜侧弱、黄昏带过渡自然。
- 保持性能稳定，不引入新的加载或渲染卡顿。

## 3. 设计原则
1. 先修结构，再修风格：先解决厚度和强度，再做色彩微调。
2. 保守迭代：每轮只改一类参数，便于定位效果变化来源。
3. 可回退：每轮参数变更独立提交，保留回滚点。

## 4. 执行计划

### Phase A - 基础参数收敛（低风险）
目标：去除“描边感”。

操作方向：
- 缩小大气层半径（外壳距离地表更近）。
- 提高 Fresnel 指数，让光晕更贴边更薄。
- 下调发光强度与 alpha 上限，避免高饱和外环。

预期结果：
- 远看不再是粗蓝圈。
- 近看仍能感知边缘大气存在。

### Phase B - 昼夜分区（核心）
目标：让光晕符合光照方向。

操作方向：
- 在 atmosphere shader 中引入太阳方向参与计算。
- 日侧使用偏青白、较亮但薄的边缘。
- 夜侧明显减弱，避免整圈同亮。
- 黄昏带加入轻微暖色过渡（非常克制）。

预期结果：
- 光晕具有方向性和空间感，不再“同心圆描边”。

### Phase C - 层次微纹理（可选增强）
目标：打破“完美几何圈”的人工感。

操作方向：
- 对光晕强度加入极弱噪声扰动。
- 云层边缘做微弱耦合高光，增强体积感。
- 仅在必要时引入轻量 glow（严格控制强度）。

预期结果：
- 视觉更精致，但不会过度特效化。

## 5. 验收标准
- P0：地球边缘不再呈现明显粗蓝圈。
- P0：各视角下（默认、拖拽后）无突兀外环。
- P1：地球主体细节优先于光晕，视觉焦点正确。
- P1：本地与线上均能稳定渲染，无新增报错。
- P1：移动端不出现显著掉帧或闪烁。

## 6. 测试计划
1. 本地验证
- 连续 3 次冷启动截图（默认视角）。
- 交互后截图（拖拽+缩放）对比边缘观感。

2. 线上验证（GitHub Pages）
- 连续 3 次刷新，确认效果一致。
- 至少一次移动端视口模拟检查。

3. 降级验证
- 纹理降级模式下，光晕仍不过分喧宾夺主。

## 7. 交付方式
- 每个 Phase 独立提交（Commit），附“改前/改后”截图证据。
- 完成 v3.0 后，在 README 增加一条“视觉优化记录（Atmosphere v3）”。
